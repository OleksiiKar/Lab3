## Опис реалізації

Програма написана на мові **Java** і демонструє роботу з масивами об'єктів, сортування за декількома критеріями та юніт-тестування.

### 1. Клас Airplane

Об'єктна модель літака містить 5 полів:

* `Model` (String) — назва моделі літака;
* `Manufacturer` (String) — назва виробника;
* `Top Speed` (Integer) — максимальна швидкість літака (км/год);
* `Max Altitude` (Integer) — максимальна висота польоту (м);
* `Payload` (Double) — масимальне корисне навантаження (тонни).

Реалізовано метод `equals()` для пооб'єктного порівняння всіх полів та перевизначено `ToString()` для зручного виводу інформації про об'єкт.

### 2. Клас `Main`: Виконавчий клас

* Створення масиву: Ініціалізація масиву об'єктів Airplane тестовими даними.
* Сортування за максимальною швидкістю з використанням .sort разом із Comparator.comparingInt та .reversed для впорядкування
літаків від швидших до повільніших.
* Сортування за максимальною висотою польоту з використанням .sort разом із Comparator.comparingInt для впорядкування
літаків від нижчих до вищих.
* Перезапис стандартного методу `equals()` для порівняння об'єктів класу `Airplane`

### 3. Тестування (Unit Tests) перевіряють:

* Коректність виводу об'єкту.
* Коректність роботи getters та setters.
* Роботу методу порівняння `equals`.
